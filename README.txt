---AVR Training---

Task 0:

Download the ATmega4809 datasheet: https://www.microchip.com/en-us/product/ATMEGA4809#document-table
Optional web alternative: https://onlinedocs.microchip.com/?find=4809&doctype=Datasheet

Download the ATmega4809 Curiosity Nano Hardware User Guide: https://www.microchip.com/en-us/development-tool/DM320115
Optional web alternative: https://onlinedocs.microchip.com/?find=4809%20curiosity&doctype=User%20Guide

Download the Curiosity Nano Base Hardware User Guide: https://www.microchip.com/en-us/development-tool/AC164162
Optional web alternative: https://onlinedocs.microchip.com/?find=curiosity%20nano%20base&doctype=User%20Guide

If you're using MPLAB X, then you also need to download the XC8 compiler, which can be found here: https://www.microchip.com/en-us/tools-resources/develop/mplab-xc-compilers#Downloads
