---Session 1---


Task 1:

Learn how to open MPLabX or Microchip Studio.
Read the code, what do you think it does?

Upload the code to see what it does!


Task 2:

Set up a button with pull-up. Use this to control the LED.


Task 3:

Add additional logic so that the button toggles the LED.


Task 4: (OPTIONAL)

Add the OLED board.
Find out where the buttons and LEDs connect.
Do something fun with the additional LEDs and buttons.
